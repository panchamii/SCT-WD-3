# Quiz Web Application – Interactive Quiz Web Application

QuizBloom is a modern and interactive quiz web application built using HTML, CSS, and JavaScript.  
It allows users to answer different types of quiz questions and receive instant feedback along with a final score summary.

## Features
- Interactive quiz interface
- Multiple question types:
  - Single Choice
  - Multiple Select
  - True/False
  - Fill in the Blanks
  - Order/Drag and Drop
- Real-time feedback for answers
- Timer for each question
- Score tracking system
- Final result summary with statistics
- Responsive and visually appealing UI

## Technologies Used
- HTML
- CSS
- JavaScript
- Google Fonts

## Functionalities
- Start quiz from a splash screen
- Navigate through questions
- Submit answers and view feedback
- Track score progress
- Display final results and performance statistics
- Restart quiz option

This project was developed to practice:

- Interactive UI Design
